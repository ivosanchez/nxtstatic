# List

## Component

### `<app-list>`

## API

### Unordered List

|  Prop  |   Type   | Default | Description                           |
| :----: | :------: | :-----: | ------------------------------------- |
| `type` | `String` |  `ul`   | Defines an unordered (bulleted) list. |

### Ordered List

|  Prop  |   Type   | Value | Description              |
| :----: | :------: | :---: | ------------------------ |
| `type` | `String` | `ol`  | Defines an ordered list. |

## Examples:

### Unordered List

```html
<app-list type="ul">
  <li>ITEM 1</li>
  <li>ITEM 2</li>
  <li>ITEM 3</li>
</app-list>
```

### Ordered List

```html
<app-list type="ol">
  <li>ITEM 1</li>
  <li>ITEM 2</li>
  <li>ITEM 3</li>
</app-list>
```
